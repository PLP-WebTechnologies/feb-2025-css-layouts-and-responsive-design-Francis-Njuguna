### Breakpoints
- Mobile: < 480px
- Tablet: 480px - 768px
- Desktop: > 768px

### Styling Features
- Modern color scheme (#2563eb, #4b5563)

- Box shadows and transitions

- Text sizes adjust for readability

- Media query compatibility
